# Fer el exercici següent

Crea una llista de noms que gestionarem amb un menú, amb les seguents opcions
Llistar noms (mostrarà tots els noms fins al moment, al moment de llistar-lo, els ha de mostrar amb majuscules i a mes al canto, el numero de lletres que tinguis, si no tenen infirmació o estan en blanc no s'han de mostrar) . Al finalitzar el llistat t'ha de mostrar el numero de noms existents.
Afegir nom(Buscara la primera posició no informada  i afegira el nom introduit al usuari)
Modificar nom(Buscara el nom i el substituira pel nou que introdueixis, si no el troba ho ha de dir)
Esborrar nom(opció fàcil, quant trobes la posició la deixes en blanc o a null, el que prefereixis).
